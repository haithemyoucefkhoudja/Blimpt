## Installation

### System Requirements
- At least 4GB of RAM
- 500MB of free disk space
- For optimal performance: 8GB of RAM and a modern multi-core processor

### Windows
1. Download the latest version for Windows [here](https://github.com/haithemyoucefkhoudja/QUICK-GPT-TAURI/releases/latest)
2. Run the installer and follow the on-screen instructions
3. Launch Quick-GPT using the desktop shortcut or from the Start menu
